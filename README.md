# pyBackEnd

## How to run

### 1) Download python
#### :bookmark_tabs: [Python 3.7.7](https://www.python.org/downloads/release/python-379/)

### 2) Instal or uppdate pip
   Write in cmd:
   `python -m pip install --upgrade pip`
   
### 3) Instal pipenv
   Write in cmd:
   `$ pip install --user pipenv`
   
### 3) Instal Flask
   Write in cmd:
   `$ pip install Flask`
   
## Now you should be able to run the project
